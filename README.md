
#  Backend Dart

Projeto feito para estudo da linguagem dart para o bakcend.




## Funcionalidades

- Autenticação usando o pradrão JWT
- CRUD simples com dados de um user - nome, email, password e role



## Stack utilizada

**Documentation:** Swagger 

**Back-end:** Dart

**Data Base:** Postgres

**ORM:** Prisma
